Project Overview

This LAN design project demonstrates understanding of enterprise-level network architecture principles. The simulation includes multiple departments with separate network segments, implementing proper VLAN segregation for improved security and network management.

Design Highlights:

  • Multi-tier network architecture with core, distribution, and access layers
  • OSPF dynamic routing for optimal path selection
  • STP implementation for loop prevention
  • VLAN implementation for departmental network segmentation
  • Access Control Lists (ACLs) for network traffic filtering
  • DHCP server configuration for automatic IP assignment

Technical Challenges Addressed:

  • Optimizing bandwidth utilization across network segments
  • Ensuring redundancy and failover capabilities
  • Implementing inter-VLAN routing correctly
  • Configuring security policies without limiting legitimate traffic
  • Scaling the network to support future growth

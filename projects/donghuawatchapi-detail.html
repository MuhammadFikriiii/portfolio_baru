DonghuaWatch API

2026 Backend API

DonghuaWatch API adalah RESTful API yang kuat dan efisien yang dirancang untuk menyediakan data streaming Donghua dari berbagai sumber. API ini mendukung fitur pencarian, filter berdasarkan genre/status, serta sistem caching yang canggih untuk memastikan performa yang cepat dan handal bagi aplikasi frontend.

Tech Stack

  • Backend: Node.js, Express
  • Scraping: Cheerio, Axios
  • Styling: CSS for API Docs
  • Caching: Memory Cache, Vercel Edge
  • Infrastructure: Vercel Platform

API Features

  • ✓ Multi-source Scraping (Optimized)
  • ✓ Image Proxy & Hotlink Shield
  • ✓ Request Collapsing & Debouncing
  • ✓ JSON Schema Validation
  • ✓ Detailed Error & Success Responses

Performance

  • Response Time: < 150ms
  • Uptime: 100% (SLA)
  • Edge Delivery: Global CDN
  • Stateless: Serverless Ready

Project Overview

DonghuaWatch API dikembangkan untuk menjadi tulang punggung (backbone) dari ekosistem DonghuaWatch. Tujuan utamanya adalah untuk menyatukan data dari berbagai sumber ke dalam satu format JSON yang konsisten dan mudah dikonsumsi oleh aplikasi mobile maupun web.

Technical Implementation:

  • Implementasi scraper menggunakan Cheerio untuk mengekstrak data dari HTML secara efisien.
  • Sistem caching bertingkat (Memory & Edge) untuk mengurangi beban pada sumber data asli dan mempercepat loading.
  • Fitur image proxy untuk menangani masalah hotlinking dan meningkatkan performa pemuatan gambar di frontend.
  • Penerapan middleware untuk validasi request dan logging.
  • Arsitektur serverless yang memungkinkan skalabilitas otomatis sesuai trafik.

Key Endpoints:

  • /api/v1/home - Mengambil data terbaru dan terpopuler.
  • /api/v1/search?q=... - Mencari judul donghua spesifik.
  • /api/v1/detail/:id - Informasi mendalam tentang satu judul.
  • /api/v1/episode/:id - Mendapatkan link streaming video.

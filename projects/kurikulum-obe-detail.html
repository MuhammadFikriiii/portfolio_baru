Kurikulum OBE Poliban

Enterprise Resource Planning

2024 Poliban Core Governance

Sistem Informasi Penyusunan Kurikulum Poliban Berbasis OBE adalah platform digital yang dirancang untuk mengintegrasikan proses perencanaan, pemetaan, dan evaluasi kurikulum perguruan tinggi sesuai standar Outcome-Based Education.

Tech Stack

  • Backend: Laravel 10 (PHP)
  • Frontend: Tailwind CSS
  • Database: SQL Server / MySQL
  • Engine: Blade Template Engine
  • Cloud: VPS Poliban Deployment

Key Features

  • ✓ Matriks Pemetaan CPL ke MK
  • ✓ Modul Penginputan Bahan Kajian
  • ✓ Generasi Laporan RPS Otomatis
  • ✓ Dashboard Statistik Kurikulum
  • ✓ Manajemen Role Admin & Dosen

Project Status

  • Duration: 4 Months
  • Authority: Official Poliban
  • User Base: Seluruh Dosen
  • Compliance: Standar BAN-PT

Project Overview

This project was developed as part of the curriculum modernization initiative at Politeknik Negeri Banjarmasin. The system facilitates the transition to Outcome-Based Education (OBE) methodology, enabling institutional stakeholders to systematically map learning outcomes, courses, and assessments.

Key Responsibilities:

  • Backend development using Laravel framework
  • Database design and optimization for complex data relationships
  • Frontend interface implementation with Tailwind CSS
  • User authentication and role-based authorization
  • Testing and deployment to production environment

Impact & Results:

  • Reduced curriculum planning time by 60%
  • Improved transparency in learning outcome alignment
  • Enabled data-driven curriculum decisions
  • Increased institutional capacity for accreditation requirements
  • Successfully adopted by 50+ faculty members

SIM-A BPKPAD

Electronic Archive Management System

2024 Banjarmasin Gov Internal Tools

SIM-A (Sistem Informasi Manajemen Arsip) adalah solusi enterprise yang dikembangkan untuk BPKPAD Kota Banjarmasin guna mendigitalisasi dan mengoptimalkan pengelolaan dokumen keuangan dan aset daerah secara aman dan terintegrasi.

Tech Stack

  • Backend: Laravel 10
  • Frontend: Tailwind CSS
  • Database: MySQL / MariaDB
  • Security: AES-256 Encryption
  • Output: PDF Generation (DomPDF)

Core Features

  • ✓ Digitalisasi Dokumen Fisik
  • ✓ Klasifikasi Arsip Otomatis
  • ✓ Pencarian Cepat (Fast Search)
  • ✓ Hak Akses Berjenjang (ACL)
  • ✓ Riwayat Aktivitas (Audit Log)

Project Stats

  • Duration: 5 Months
  • Deployment: Local Intranet
  • Users: BPKPAD Officers
  • Efficiency: +85% Speed

Project Overview

This project addressed the critical need for modernized archive management at BPKPAD. The previous manual system was inefficient, insecure, and non-compliant with government digitalization requirements. The new system provides a centralized, secure, and efficient platform for managing institutional documents.

Key Responsibilities:

  • Backend development using Laravel framework
  • Database design for efficient document management
  • Frontend implementation with responsive UI
  • User authentication and role-based access control
  • Document upload and storage management
  • Search functionality implementation
  • Audit logging for compliance tracking
  • System testing and deployment

Business Impact:

  • Reduced document retrieval time by 80%
  • Improved document security and access control
  • Enhanced compliance with government regulations
  • Eliminated physical storage space requirements
  • Enabled better institutional record management
  • Increased staff productivity and efficiency
